### Function
* 声明
    * function foo(){}
    * var foo = new Function("a", "b", "console.log(1)");
* 表达式
    * 变量函数表达式：var foo = function(){};
    * 命名函数表达式：function foo(){};
    * 即调函数表达式：
        * (function(){})();
        * (function(){}());
        * !function(){}();
        * new function(){}();
    * 首类函数表达式：return function(){};
* 参数
    * a = a || 1;
    * var a = arguments[0] ? arguments[0] : 1;
* 内容类型
    * 无返函数：function foo(){}
    * 有返函数：function foo(){ return 1 }
    * 内调函数：
```
function foo(x){
  if(x > 2){
    return foo(x-1) + foo(x-2);
  }else{
    return 1;
  }
}
foo(3);
```
    * 闭包函数：
```
function foo(x){
  return function(){
    return x++;
  };
}
var test = foo(1);
test();
```
    * 构造函数：
```
function Foo(){
  this.one = 1;
}
var test = new Foo();
```
* 方法
    * Function.prototype.call(obj, "a", "b");
    * Function.prototype.apply(obj, ["a", "b"]);
    * Function.prototype.bind(obj, "a", "b");
